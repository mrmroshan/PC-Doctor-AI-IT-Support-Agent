# PC Doctor v1.0.0

Initial public release of `PC Doctor - AI IT Support Agent`.

## Highlights

- Supervised AI-driven IT diagnostics and guided remediation workflow
- Safer execution model with explicit `YES`, `SKIP`, `ABORT`, and high-risk confirmation gates
- Expanded project governance with clear contributing and security guidance
- License migrated to AGPL-3.0 for open-source collaboration with stronger reciprocity protections

## Included in This Release

- Core launcher and diagnostics scripts:
  - `install.bat`
  - `diagnose.ps1`
- Documentation and project metadata updates:
  - `README.md`
  - `LICENSE`
  - `NOTICE`
  - `CONTRIBUTING.md`
  - `SECURITY.md`
  - `CHANGELOG.md`
